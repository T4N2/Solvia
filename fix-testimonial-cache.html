Opsi 1: Hard Refresh (Paling Mudah)

Opsi 2: Clear Browser Cache

2. Pilih "Cached images and files"

3. Klik "Clear data"

4. Refresh halaman

Opsi 3: Disable Cache (Developer)

1. Buka Developer Tools (F12)

2. Pergi ke tab "Network"

3. Centang "Disable cache"

4. Refresh halaman
